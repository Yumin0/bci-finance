# 品牌色彩規範

> 來源：公司提供之品牌用色規範文件
> 最後更新：2026-07-09

---

## 主要色彩

確保我們的色彩與品牌其他元素一樣具有辨識度是重要的。但這些色彩應當以我們的主要色彩為主，其他色彩則為輔。

| 預覽 | HEX |
|------|-----|
| 🟨 | `#FFEA41` |
| ⬛ | `#111214` |
| ⬜ | `#FFFFFF` |

## 輔助色彩

我們的輔助色彩中的所有顏色都要能與標誌色相輔相成，同時為各類傳播內容提供靈活運用的空間。輔助色彩應用於需要多種顏色的場合，在任何情況下，都要力求讓主要色彩保持最高的比重和重要性。

| 預覽 | HEX |
|------|-----|
| ⬜ | `#F2F2F3` |
| 🔲 | `#D4D8E3` |
| ⬜ | `#9599A4` |
| 🟦 | `#ADCFFF` |
| 🟪 | `#B9ADFF` |
| 🟩 | `#C0EDE2` |

## 品牌用色側重規範

為了在品牌傳播中實現一致的視覺效果，色彩的運用應以主要色彩系列為重點，並在適合的情況下輔以輔助色彩系列作為支持。
