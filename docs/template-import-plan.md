# 筑今系統 → BC資金分配系統 共用範本匯入確認清單（v2）

- 資料來源：筑今系統（bci-financial.com）諮詢議會頁，2026 年第 1～28 週，共 **407 筆**申請單
- 篩選標準：同課別＋同費用項目＋項目名稱（去除月份日期）相同，**出現 3 週以上**視為例行申請 → 共 **25 組**
- 其中 **7 組**屬舊組織單位（第5A課、第17課 集客行銷課、第16課 品牌策略與經營課）不移轉、**3 組**（清潔費）staging 已有範本略過 → **擬新建 15 個共用範本**
- 每組取最近一筆已核准/已付款單據的完整內容作為範本值
- 產出日期：2026-07-11（v2：依 Yumin 指示調整）

## 全域預設規則

| 規則 | 說明 |
|---|---|
| 憑證類型 | 舊系統無此欄位，一律預設「國內公司」 |
| 幣別 | 依舊單帶入（全部為台幣） |
| 職務（apply_role） | 留空，由申請人自選（同現有範本作法） |
| 金額 | **固定費用**（28 週內金額幾乎不變）帶入預設值；**浮動費用**留空，申請人每次自填 |
| 項目/摘要中的月份日期 | 改成「X月」「X/X」佔位字樣（同現有範本「X月份清潔費」作法） |
| 適用組織範圍（org_unit_ids） | 勾對應**處級**節點（整個處的成員都看得到，同現有範本作法） |
| 附件 | 範本不含附件，略過 |
